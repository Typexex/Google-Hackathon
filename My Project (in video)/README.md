# Tesseract Project

A 3D immersive landing page recreating the "Tesseract" visual concept.

## Tech Stack
- Vite + React
- Three.js (@react-three/fiber)
- TailwindCSS
- Framer Motion

## Setup & Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
